# Chapter 10 — Q&A + Open Discussion (Including Games!)

**Duration:** 30 minutes  
**Format:** facilitated discussion + group games  
**Prerequisites:** all chapters complete

---

## Structure

| Time | Activity |
|------|----------|
| 0–10 min | Open Q&A — bring your real-world questions |
| 10–25 min | Games (choose 1–2 from `games/`) |
| 25–30 min | Closing reflection |

---

## Open Q&A

Come prepared with at least one question in one of these categories:

**Technical questions:**
- How would this work on iOS?
- What happens when the app has no accessibility IDs?
- How do you handle biometric authentication in the agent loop?
- Can the agent test offline/airplane mode scenarios?

**Team and process questions:**
- How do you convince a team to trust AI-generated tests?
- What does the QA engineer's role look like in an agentic system?
- How do you handle version control for agent-generated code?
- What does PR review look like when a healing agent proposes a patch?

**Cost and scale questions:**
- What does this cost per test run?
- How do you run 1000 agent tests in CI without blowing the LLM budget?
- When is a YAML flow better than an agent run?
